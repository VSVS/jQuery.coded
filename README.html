Thanks for trying out the jQuery.coded alpha 0.1 release. As an alpha release, this will be far from perfect, and we look forward to your feedback to improve this framework. This library helps you code with JAM markup and expedite web app and website development in a neat and rapid fashion. To get started with documentation and other info, please review the index.html file in your browser. For a tutorial on JAM, please read the JamTutorial.html file. For a tutorial on jQuery.coded methods, please read the methodsTutorial.html file. For questions, please visit http://jquerycoded.com.
